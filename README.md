# v3-grid-layout

A powerful, draggable, and resizable grid layout component built natively for **Vue 3** and **TypeScript**.

[![npm version](https://img.shields.io/npm/v/v3-grid-layout.svg)](https://www.npmjs.com/package/v3-grid-layout)
[![GitHub license](https://img.shields.io/github/license/zhl1232/v3-grid-layout.svg)](LICENSE)

## 🌟 Why v3-grid-layout?

The original, heavily relied-upon `vue-grid-layout` library has been abandoned for years and completely lacks official support for Vue 3 and modern TypeScript workflows. This repository is a modernized, fully optimized, and actively maintained successor designed to bridge this critical gap in the frontend ecosystem.

It provides a seamless, type-safe upgrade path for developers and enterprises migrating complex dashboards, cloud management consoles, and customizable grid UIs to the Vue 3 ecosystem.

## ✨ Key Features

- **Vue 3 & TypeScript Native**: Built from the ground up with the Vue 3 Composition API and strict type definitions.
- **Draggable & Resizable**: Smooth, high-performance drag-and-drop and grid item resizing.
- **Responsive Layouts**: Supports fluid grid adjustments across custom device breakpoints.
- **Serialized Layouts**: Easily save, restore, and dynamically update grid state via reactive props.
- **Ecosystem Ready**: Tailored to work flawlessly with modern bundlers like Vite and modern testing frameworks.
