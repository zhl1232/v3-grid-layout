# vue-grid-layout-next

Vue 3 可拖拽、可缩放的栅格布局组件库，基于 [interact.js](https://interactjs.io/) 实现。

## 文档结构

| 章节 | 内容 |
|------|------|
| **说明** | 安装与快速上手（本页） |
| **GridLayout** | 交互式示例与 Controls 面板 |
| **API 参考** | Props、Events、Expose 完整列表 |

在 Canvas 视图中按 **`D`** 可打开 Controls 面板实时调整参数。

## 安装

```bash
npm install vue-grid-layout-next
# 或
pnpm add vue-grid-layout-next
# 或
yarn add vue-grid-layout-next
```

需要 `vue` ^3.5.0 作为 peer dependency。

## 快速上手

### 全局注册

```js
import { createApp } from 'vue'
import App from './App.vue'
import GridLayoutPlugin from 'vue-grid-layout-next'
import 'vue-grid-layout-next/dist/style.css'

createApp(App).use(GridLayoutPlugin).mount('#app')
```

### 局部注册

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { GridItem, GridLayout } from 'vue-grid-layout-next'
import type { Layout } from 'vue-grid-layout-next'
import 'vue-grid-layout-next/dist/style.css'

const layout = ref<Layout>([
  { x: 0, y: 0, w: 2, h: 2, i: '0' },
  { x: 2, y: 0, w: 2, h: 4, i: '1', static: true }
])
</script>

<template>
  <GridLayout
    v-model:layout="layout"
    :col-num="12"
    :row-height="30"
    :is-draggable="true"
    :is-resizable="true"
    :margin="[10, 10]"
  >
    <GridItem
      v-for="item in layout"
      :key="item.i"
      drag-allow-from=".handle"
      v-bind="item"
    >
      <div class="handle">拖拽手柄</div>
      <div>{{ item.i }}</div>
    </GridItem>
  </GridLayout>
</template>
```

## 主要特性

- Vue 3 Composition API + TypeScript
- 拖拽与缩放（interact.js）
- 响应式断点布局
- 单项覆盖 `isDraggable` / `isResizable`
- 固定像素尺寸（`colWidth`、`pixelWidth`、`pixelHeight`）
- 外部拖入支持（`expose` 的 `dragEvent` / `resizeEvent`）

更多 API 细节请查看 **API 参考** 章节。
